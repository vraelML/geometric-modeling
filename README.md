# Geometric Modeling Project

**Student:** Lilian Macrez

This is my project for the Geometric Modeling course. It's a C++ 3D mesh viewer using a half-edge data structure.

## Features Implemented
- Loading OBJ files
- Computing vertex and face normals
- Silhouette detection
- Triangulation (ear clipping for concave/convex faces)
- Half-edge connectivity tests
- Surface of revolution
- Mesh simplification (shortest edge collapse)
- Catmull-Clark subdivision

## How to build
Open `TD1/myproj.sln` in Visual Studio and build/run the project (tested on Windows).
